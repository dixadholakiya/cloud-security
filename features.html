Frequently Asked Questions

How does the global static caching system work?

Once you deploy your website files (such as HTML, CSS, images) to our server nodes, we distribute copies across global edge directories. Visitors automatically request pages from the server physically closest to them, speeding up delivery considerably.

Is this template compatible with standard Nginx configurations?

Yes. The structure of these HTML, CSS, and asset files is fully standard. You can deploy it to any generic Nginx or Apache server on AWS EC2 or local machines without modifications.

What are the overage bandwidth charges?

Startup and Enterprise tiers include predefined high-bandwidth limits. If your site experiences traffic peaks that exceed these, we notify you beforehand and only charge standard, transparent flat rates based on raw bytes transfer.
