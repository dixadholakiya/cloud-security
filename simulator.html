High Severity Protocol: WS-Security / SOAP XML

SOAP XML Wrapping Attack

Exploits logic flaws in web services where the signature checking engine verifies one element (using its ID tag reference) but the business application processes a duplicate unsigned element located elsewhere in the message body.
